# slack utils

A thin wrapper of slack-SDK. This wrapper provides CLI for post messages to Slack.

## Usage

### webhook-send

```shell
Usage: slackutils webhook-send [OPTIONS]

Options:
  --header TEXT
  -m, --message TEXT
  -f, --field TEXT
  --footer TEXT
  --color [good|warning|danger]
  --help                         Show this message and exit.
```
